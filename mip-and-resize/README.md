# mip-and-resize

mip-and-resize 根据手机尺寸，调节html上font-size大小

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-and-resize/mip-and-resize.js


## 示例

### 基本用法
```html
<mip-and-resize data-and="num">
    
</mip-and-resize>
```
