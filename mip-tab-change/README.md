# mip-tab-change

mip-tab-change tab切换功能

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-tab-change/mip-tab-change.js

## 示例

### 基本用法
```html
<mip-tab-change target="content" lName="" cName="">
    
</mip-tab-change>
```

## 属性

### target

说明：指向其控制显示隐藏元素的父元素
必选项：否
类型：{类型}
取值范围：{取值范围}
单位：{单位}
默认值：无

### lName

说明：记录组件子元素类名
必选项：否
类型：{类型}
取值范围：{取值范围}
单位：{单位}
默认值：无

### cName

说明：记录组件指向元素子元素类名
必选项：否
类型：{类型}
取值范围：{取值范围}
单位：{单位}
默认值：无

### id

说明：记录该组件ID
必选项：否
类型：{类型}
取值范围：{取值范围}
单位：{单位}
默认值：无

### choose

说明：记录组件可点击的部分
必选项：否
类型：{类型}
取值范围：{取值范围}
单位：{单位}
默认值：无
