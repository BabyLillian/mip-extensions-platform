
## 属性

### data-data

说明：提交签到请求时的追加数据
必选项：否
类型：json

### data-url

说明：提交签到的接口地址
必选项：是
类型：string

### data-check-url

说明：检查是否已经签到的接口地址
必选项：是
类型：string
